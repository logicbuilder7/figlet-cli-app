# Figlet CLI App 🚀

A simple Command Line Interface (CLI) application built using Node.js that converts normal text into ASCII art using the figlet package.

---

## 🎯 Project Purpose

This project was created to learn:

* How to use npm packages
* CLI-based application development
* Basic Node.js concepts

---

## 🛠️ Tech Stack

* Node.js
* figlet npm package

---

## ⚙️ Installation & Run

```bash
npm install
node index.js
```

---

## ✨ Features

* Convert text into ASCII art
* Simple CLI-based interaction
* Lightweight and fast
